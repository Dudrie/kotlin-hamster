Data to generate a tile upon initialization.

Parameters

location

HamsterLocation this tile will be at.

tileType

Type of the tile. Defaults to GameTileType.Floor.

grainCount

Initial amount of grains on this tile. Default to 0.

hideGrainCount

If true the tile should hide its grain count and only show if it has at least one grain or zero.
