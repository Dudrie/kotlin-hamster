open fun redo()

Redoes the last Command that got undone.

The function does NOT check whether a Command can be redone before executing it. It assumes that any Command that was previously successfully executed can be so again and that there are no other commands executed if there are still Commands which can be redone.

This makes sure that the redone Command is back in the list of executedCommands.
